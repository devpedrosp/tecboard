# TecBoard

Projeto desenvolvido durante o curso **"HTML e CSS: Ambiente, Estrutura e Estilo"** da plataforma **Alura**, com foco na aprendizagem dos fundamentos de desenvolvimento web, versionamento de código com Git e publicação de sites utilizando GitHub Pages.

## 🛠️ Tecnologias Utilizadas

* **HTML5** — Estruturação semântica do conteúdo.
* **CSS3** — Estilização e organização visual da interface.
* **Git** — Controle de versão e gerenciamento do histórico do projeto.
* **GitHub** — Hospedagem do repositório e publicação da aplicação.

## 🌐 Acesso ao Projeto

O projeto está disponível online através do GitHub Pages:

## 🎯 Objetivos de Aprendizagem

Durante o desenvolvimento deste projeto foram praticados os seguintes conceitos:

* Estruturação de páginas web com HTML e CSS.
* Utilização do Git para controle de versões.
* Gerenciamento de repositórios com GitHub.
* Uso de comandos básicos do terminal.
* Criação e utilização de branches e pull requests.
* Boas práticas de versionamento de código.
* Publicação de sites estáticos com GitHub Pages.
* Conceitos fundamentais de hospedagem e domínio de sites.

## 🚀 Como Executar Localmente

1. Clone o repositório:

```bash
git clone https://github.com/devpedrosp/tecboard.git
```

2. Acesse a pasta do projeto:

```bash
cd tecboard
```

3. Abra o arquivo `index.html` em seu navegador ou utilize uma extensão como o Live Server no VS Code.
